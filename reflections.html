My Goals

Throughout this course, I hope to be able to learn critical info relating to Information Technology (IT), such as how to program, develop, deploy, and maintain applications and websites, diagnose and troubleshoot issues that may arise, and obtain good coding practices such as formatting and sanitisation. I also hope to be able to learn various programming languages and understand which solution is best for issues at hand.

What Have I Learned?

I find the most challenging thing about programming is that there is a lot of troubleshooting involved, and because a single incorrect syntax or typo can completely break the whole thing, it is quite hard to figure out which line out of hundreds of lines of code is causing issue.

I also found that the easiest (and organic) way to learn programming is to have an issue that you wish to solve, as learning about what Python is able to do is not very useful without having a real-world situation to apply it to.

YouTube and Stackoverflow are very valuable websites to gain ideas on how to implement solutions.
