I found that the easiest way for me to learn programming and absorb information is to have an issue that I wish to solve, as learning about what Python is able to do is not very useful without having a real-world situation to apply it to.

YouTube and StackOverflow are very valuable websites to get solutions.

Version control and backups are important not only because you might lose your data, but because you might do hours of work only to realise it didn't solve your problem and you wish to delete everything start over.

Encryption isn't just to keep private date safe from prying eyes, it's also important to guarantee the data has not been modified by a malicious third party.

Open source might be good because they're budget friendly and anyone is free to poke around for vulnerabilities, but proprietary solutions have their own merit - proprietary ones tend to be more optimised, updated more frequently, and have more and/or better features.

The biggest threats to privacy are not individuals, but instead overly nosy companies and organisations.

Seems like there are a lot of programming languages that one must be familiar with in web development.

HTML and CSS

Pretty straightforward to understand; almost in plain English.

Downside: Good luck with fancy formatting.

Python

The best way to learn Python, it seems, is to have a goal that you wish to work towards, or 'dive in' as the saying goes. Reading through documentation usually is not productive.

Python's built-in error handling is vague, the language unclear and I find not very useful.

There are a bunch of challenges on the Web such as attempting to print 1 to 100 without typing in any numbers or using loops. I don't know how anyone manages to come up with something that satisfies the requirement within 20 lines of code.

SQL

SQL looks a powerful tool for how simple it is to get into and I should definitely learn more about it, although keeping track of thousands of entries seems rather daunting.

The possibility of accidentally destroying your entire database because of one typo seems like an oversight.

JSON and Javscript

Apparently many sites on the Web use these two for scripts, and I should also schedule time to get into it.
